Run on each cron trigger:

1. cd to repo root (growth-signal-outbound, main branch)
2. python3 -m venv .venv 2>/dev/null || true
3. .venv/bin/pip install -q -r requirements.txt
4. Verify queue exists: ls -la send_queue.json (must be at repo root)
5. Run: bash scripts/cloud-send.sh

COMPOSIO_API_KEY must be in Cloud Agent secrets.

Report from send_queue.json:
- sent this run (status -> "sent")
- still pending + next scheduled_at

If send_queue.json is missing, run: .venv/bin/python run.py schedule
